Overview:-
This project presents a comprehensive Business Intelligence (BI) solution for analyzing insurance data using an interactive dashboard. The system transforms raw insurance datasets into meaningful insights through data modeling, visualization, and analytical reporting.The dashboard enables stakeholders to monitor policy performance, claims trends, customer segmentation, and feedback analysis in real time.
Insurance companies deal with large volumes of data related to policies, claims, and customers. Extracting actionable insights manually is difficult and time-consuming.
This project aims to:-
1)Simplify complex data analysis.
2)Provide real-time insights.
3)Support data-driven decision-making.
📊 Insights & Analysis:-
1) Travel policies generate the highest premium revenue.
2) Majority of claims are successfully settled.
3) Adults have the highest claim contribution.
4)Positive customer feedback dominates.
5)Claim rejection rate is comparatively low.
 Business Impact:-
1)Improves decision-making for insurance companies.
2)Helps identify high-risk policy segments.
3)Enhances customer experience analysis.
4)Enables performance monitoring.
